# Flutter

This is the Flutter version of Shader Gadgets. It uses Flutter widgets for the
UI and `flutter_shaders` for the Glide fragment shader.

## Setup

```sh
cd flutter
flutter pub get
flutter run
```

Select an iOS simulator or connected device when prompted.

This repository does not include the Flutter Android project. Add it only when
Android builds are needed:

```sh
flutter create . --platforms=android
```
