# Web

This is the React and WebGPU version of Shader Gadgets. It uses Vite for local
development and TypeGPU for the shader pipeline.

## Setup

Use a browser with WebGPU support.

```sh
cd web
npm install
npm run dev
```

The local server runs at `http://localhost:5173/` by default.

## Build

```sh
cd web
npm run build
```

To preview the production build:

```sh
npm run preview
```
